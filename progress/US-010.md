### Learnings for future iterations:

**PyQt6 Type Checking:**
- Use `# type: ignore[reportUnknownMemberType]` for PyQt6 signal connections (`.connect()`)
- Use `# type: ignore[reportMissingSuperCall]` for classes without explicit base class `__init__`
- QMenu.addAction() and signal connections trigger type warnings that need suppression
- Lambda arguments for signal handlers need placeholder prefix (`_checked`) to avoid ARG005

**MenuBar/System Tray Patterns:**
- QSystemTrayIcon provides cross-platform system tray support
- QMenu for context menu, QAction for menu items
- Use `setEnabled(False)` for display-only menu items (status labels)
- Load icons from assets/ directory with fallback to system icons
- QTimer for periodic UI updates (status, statistics)
